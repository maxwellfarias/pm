## QUADRO-RESUMO FINAL: O QUE A BANCA ADORA COBRAR

| Tema | Pegadinha Classica |
|---|---|
| Prioridade no atendimento | **ENVIAR VIATURA PRIMEIRO**, coletar infos depois |
| Vitima liga | **NUNCA** retornar a ligacao |
| Vitima nao e dispensada | Atendente **NUNCA** deve dispensar a vitima |
| Crianca no local | Acionar **CONSELHO TUTELAR** |
| PM obrigatoria | PM deve ser acionada com **URGENCIA** |
| Guarnição de resgate | AR so chega ao local **COM PRESENCA DA PM** |
| Socorro disfarcado | Perguntas de **SIM ou NAO** (8 perguntas) |
| Orientacao ao ligante | Orientar que **FALE BAIXO** |
| Descricao do agressor | So colher se quem ligou **NAO** e a vitima |
| Tom de voz | **NUNCA** usar tom inquisitivo ou rispido |
| Minimizar ocorrencia | **NUNCA** classificar como menor importancia |
| Julgar a vitima | **NUNCA** julgar ou perguntar "por que nao denunciou" |
| Ameacar prisao | **NUNCA** sugerir prisao (desencorajaria futuros pedidos) |
| Comentarios pessoais | **NUNCA** fazer comentarios depreciativos ou julgamentos |

---

> **Dica de estudo:** Este protocolo e muito objetivo — decore: (1) ENVIAR viatura PRIMEIRO; (2) NUNCA retornar ligacao da vitima; (3) AR so com PM; (4) crianca = Conselho Tutelar; (5) as 8 perguntas sim/nao do socorro disfarcado; (6) as 8 vedacoes do atendente. Essas sao as pegadinhas recorrentes!
