## QUADRO-RESUMO FINAL: O QUE A BANCA ADORA COBRAR

| Tema | Pegadinha Classica |
|---|---|
| Feminicidio | Agora e art. **121-A** (autonomo), nao mais qualificadora do 121 |
| Feminicidio pena | **20-40 anos** (maior que homicidio qualificado: 12-30) |
| Lesao "gravissima" | O Codigo **NAO** usa esse termo — e doutrina |
| Lesao grave vs gravissima | Grave = PIDA / Gravissima = PEIDA (mnemonicos) |
| Viol. domestica §9o | Pena atualizada: **2-5 anos reclusao** (Lei 14.994/2024) |
| Lesao seguida de morte | Crime **preterdoloso** (dolo lesao + culpa morte) |
| Ameaca contra mulher | Pena em **dobro** + acao penal **incondicionada** |
| Perseguicao (stalking) | Acao penal por **representacao** |
| Viol. psicologica 147-B | So vitima **mulher** + crime **subsidiario** |
| Viol. psicol. + IA | Aumento de **metade** se uso de IA (Lei 15.123/2025) |
| Estupro | Tipo unico desde 2009 (conj. carnal + ato libidinoso) |
| Inst. ensino (2025) | Nova agravante (61,II,m) + aumento no homicidio e lesao |
| Homicidio culposo | **DETENCAO** (nao reclusao!) |
| Perdao judicial | Homic. culposo §5o: juiz pode deixar de aplicar pena |
| Agravantes art. 61 | So incidem se NAO sao elementares/qualificadoras |
